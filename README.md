# ☁️ My Cloud Security Portfolio

A hands-on collection of cloud security projects demonstrating practical experience in AWS security, Infrastructure as Code (IaC), Identity and Access Management (IAM), DevSecOps, and cloud security assessments.

This portfolio is designed to showcase real-world cloud security engineering skills through project-based learning rather than theoretical examples. Each project focuses on solving practical security challenges commonly encountered in modern AWS environments.

---

## 📖 About This Portfolio

Cloud security requires much more than understanding AWS services. Security engineers must be able to:

- Design secure cloud infrastructure
- Implement least-privilege IAM architectures
- Secure CI/CD pipelines
- Assess cloud environments for security risks
- Document findings and remediation plans
- Automate security operations

This repository brings together multiple projects that demonstrate these capabilities using AWS services, Terraform, GitHub Actions, and open-source security tools.

---

# 🏗 Portfolio Projects

| Project | Description | Technologies |
|----------|-------------|--------------|
| 🔐 **AWS Cross-Account IAM Access** | Designed and implemented secure cross-account access using IAM Roles, AWS STS, and least privilege principles. | AWS IAM, STS, Terraform |
| 🌐 **AWS VPC Infrastructure as Code** | Built a secure AWS networking environment using Terraform including VPC, public/private subnets, routing, NAT Gateway, and security groups. | AWS VPC, Terraform |
| 🚀 **Secure CI/CD Pipeline with GitHub Actions** | Developed a secure CI/CD pipeline implementing automated testing, code quality checks, container security scanning, and infrastructure validation. | GitHub Actions, Docker, Terraform |
| 🛡 **AWS Cloud Security Audit with Prowler** | Performed a comprehensive AWS security posture assessment using Prowler, analyzed findings, prioritized risks, and created a remediation roadmap. | AWS, Prowler, PowerShell |
